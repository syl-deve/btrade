# CLAUDE.md

AI 코딩 어시스턴트를 위한 구현 상세 가이드입니다. 프로젝트 개요·전략·설치는 README.md를 참조하세요.

## Added Memories
- 사용자와의 상호작용에서 git 커밋 메시지에 한글 등 비 ASCII 문자가 포함될 경우, 셸 오류를 방지하기 위해 임시 파일을 만들고 `git commit -F <file>`을 사용해야 합니다. 커밋 후에는 임시 파일을 삭제합니다. 이 방법은 Windows 환경에서 `git commit -m` 사용 시 발생하는 문제를 해결하기 위한 것입니다.
- 커밋하고 푸시하는 과정은 묻지 말고 진행해. 그리고 명령어는 한줄로 연결해서 한번에 실행해.
- 커밋하거나 파일 수정 시 UTF-8로 작성해서 텍스트가 깨지지 않도록 해.
- 문서 작성 시 포맷을 항상 마크다운 .md 형태로 해줘. 과도한 이모티콘은 지양하고 체크박스 등 필수 요소만 사용해.
- 문서를 업데이트 시 관련된 다른 모든 문서를 확인하고 영향이 있는 내용들도 같이 수정해.
- 구성도나 다이어그램을 그릴때 아스키아트 말고 머메이드 코드로 그려줘.
