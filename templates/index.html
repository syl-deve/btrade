BITRADE GUIDE

사용 설명서

처음 사용할 때

  1. 잔고와 감시 코인 확인: 상단에서 사용 가능 원화, 보유 수량, 감시 대상 코인, 현재 단계를 확인합니다.
  2. 거래 설정 열기: `거래 설정`에서 매수 기준, 익절, 손절, 필터, 리스크 제한을 조정합니다.
  3. 소액으로 시작: 1차 매수 비율을 낮게 두고 실제 체결 흐름을 먼저 확인합니다.
  4. 시작 버튼: 설정을 저장한 뒤 `시작`을 누르면 봇이 조건을 감시합니다. 정지하면 신규 매수와 자동 매매가 중단됩니다.
  5. 기록 확인: 하단 거래 내역에서 매수/매도, 체결가, 수수료, 순손익을 확인합니다.
이 대시보드는 자동 매매를 돕는 도구입니다. 시장 급변, 거래소 오류, 네트워크 지연이 발생할 수 있으므로 감당 가능한 금액으로만 운용하세요.

화면 구성

상단 상태

봇 상태, 사용자, 시작/정지, 언어 전환, 로그아웃을 제공합니다.

잔고 영역

사용 가능 원화, 현재 보유 수량, 현재 단계, 쿨다운 남은 시간을 보여줍니다.

실시간 수익률 바

보유 중인 포지션의 수익률, 평가 손익, 평균 매수가, 현재가, 손절선을 표시합니다. 보유 중이면 즉시 매도 버튼이 나타납니다.

차트

BTC 15분봉과 볼린저 하단, 누적 실현 손익 그래프를 함께 확인합니다.

지표 카드

RSI, 평균 손익, 오늘 손익, 마지막 거래, MACD, 거래량, 볼린저 상태를 요약합니다. 각 카드의 `?` 버튼을 누르면 세부 설명이 나옵니다.

거래 내역

전체 체결 내역과 순손익을 시간순으로 확인합니다.

기본 매매 흐름

  1. 스캔: 봇이 실행 중이고 보유 수량이 없으면 RSI와 선택한 필터를 감시합니다.
  2. 1차 매수: RSI가 1차 기준 이하이고 활성화된 필터를 통과하면 설정한 1차 매수 비율만큼 매수합니다.
  3. 2차 매수: 1차 매수 후 RSI가 2차 기준 이하로 더 내려가면 남은 금액으로 추가 매수합니다.
  4. 익절 대기: 목표 익절율에 도달하면 트레일링 익절이 활성화됩니다. 고점 대비 트레일링 오프셋만큼 되돌리면 매도합니다.
  5. 추세 보유: 추세 이탈 방어가 켜져 있고 EMA, 샹들리에 스탑, MACD 조건이 강하면 바로 팔지 않고 보유를 연장합니다.
  6. 손절: ATR 동적 손절이 켜져 있으면 변동성 기반 손절선을 사용하고, 실패 시 고정 손절율을 안전망으로 사용합니다.

전략 설정 설명

RSI 1차/2차 매수 기준: 낮을수록 보수적, 높을수록 공격적입니다.

목표 익절율: 이 수익률 이상에서 트레일링 익절 감시가 시작됩니다.

트레일링 오프셋: 고점 수익률에서 얼마만큼 밀리면 매도할지 정합니다.

1차 매수 비율: 첫 진입에 사용할 원화 비율입니다. 0.6이면 60%를 1차에 사용합니다.

볼린저밴드 필터: 가격이 하단 밴드 근처일 때만 매수해 과매도 구간을 노립니다.

MACD 반전 필터: 하락 모멘텀이 약해지는 반전 신호를 확인합니다.

거래량 급증 필터: 평균 거래량 대비 강한 거래량이 있을 때만 진입합니다.

ATR 손절 배수: 변동성이 클수록 손절 폭을 넓힙니다. 값이 작으면 빠르게 손절하고, 크면 흔들림을 더 허용합니다.

거래량 기준 배수: 거래량 필터가 요구하는 강도를 정합니다.

추세 이탈 방어: 익절 조건 도달 후 추세가 살아 있으면 조기 매도를 늦춥니다.

EMA 기간 / 샹들리에 ATR / MACD 약세 봉 수: 추세 유지와 약화 판정을 조정하는 고급 설정입니다.

리스크 관리와 주의사항

일일 손실 한도: 당일 실현 손실이 한도를 넘으면 봇이 자동 정지합니다. 음수로 입력합니다.

연속 손절 허용 횟수: 최근 매도 N회가 모두 손실이면 쿨다운에 들어갑니다.

쿨다운 시간: 연속 손실 후 신규 매수를 쉬는 시간입니다.

즉시 매도: 보유 중인 포지션을 수동으로 정리합니다. 긴급 상황에서만 사용하세요.

시작/정지: 정지 상태에서는 새 자동 매매가 진행되지 않습니다. 설정 변경 후 재시작 여부를 확인하세요.

권장 운영: 처음에는 보수적인 RSI, 낮은 1차 매수 비율, 일일 손실 한도, 쿨다운을 함께 켜고 운용 기록을 확인하세요.

Quick Start

  1. Check balances: Confirm KRW balance, held coin amount, monitored coin, and current phase.
  2. Open settings: Use `TRADE_CONFIG` to adjust entries, take-profit, stops, filters, and risk limits.
  3. Start small: Use a lower first-buy ratio until you understand the fill behavior.
  4. Start the bot: Save settings, then press `START`. `STOP` pauses new automated trading.
  5. Review history: Check executed buys/sells, fees, and net P&L in trade history.
This dashboard assists automated trading. Market volatility, exchange issues, and network delays can still occur. Use only capital you can afford to risk.

Trading Scenario

  1. Scan: When running with no position, the bot watches RSI and enabled filters.
  2. First buy: If RSI reaches the first threshold and filters pass, the bot buys using the first-buy ratio.
  3. Second buy: If RSI falls to the second threshold after entry, the bot adds with the remaining allocation.
  4. Take-profit watch: Once target profit is reached, trailing take-profit starts. A pullback by the offset sells.
  5. Trend hold: If trend exit is enabled and trend remains strong, the bot can delay selling.
  6. Stop-loss: ATR dynamic stop uses volatility. Fixed stop-loss is the fallback when ATR fails.

Strategy Settings

RSI thresholds: Lower is more conservative; higher is more aggressive.

Target take-profit: Starts trailing exit monitoring after this profit rate.

Trailing offset: Defines how much pullback from peak profit triggers a sell.

First-buy ratio: KRW allocation for the first entry. 0.6 means 60%.

Bollinger filter: Prefers oversold areas near the lower band.

MACD filter: Confirms weakening downside momentum.

Volume filter: Requires a stronger-than-average volume move.

ATR multiplier: Controls volatility-based stop width.

Volume multiplier: Controls how strict the volume surge requirement is.

Trend exit: Delays early selling when trend remains strong after profit target.

Risk Controls

Daily loss limit: Stops the bot when realized losses exceed the configured negative amount.

Consecutive loss limit: Enters cooldown after the latest N sells are all losses.

Cooldown: Pauses new buys after repeated losses.

Instant sell: Manually exits the current position. Use for urgent situations.

Start/Stop: Stopped state prevents new automated trades. Check state after changing settings.

Suggested operation: Start with conservative RSI, lower first-buy ratio, daily loss limit, and cooldown enabled.
